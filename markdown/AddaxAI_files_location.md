## Where to find the AddaxAI installation files?
All files are located in one folder, called `AddaxAI_files`.

<details>
<summary>Windows</summary>
<br>
  
```r
─── 📁Users
    └── 📁<username>
        └── 📁AddaxAI_files
```
</details>

<details>
<summary>macOS</summary>
<br>
  
```r
─── 📁Applications
    └── 📁.AddaxAI_files
```
</details>

<details>
<summary>Linux</summary>
<br>
  
```r
─── 📁home
    └── 📁<username>
        └── 📁.AddaxAI_files
```
</details>
